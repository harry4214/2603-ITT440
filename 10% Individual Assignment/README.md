# 2603-ITT440
## 10% Individual Assignment
Due on Week04
### Requirement
Programming language: Python
You must use:
At least one concurrent technique (e.g. threading, asyncio)
At least one parallel technique (e.g. multiprocessing)
Each student must submit original work
Code must be properly structured and commented
### Instruction
You are required to complete an individual assignment based on the topic of Parallel Programming.

Each student must propose a unique project title. No two students are allowed to work on the same problem or application.

Suggest a unique title and problem statement related to parallel programming.
The proposed topic must be approved by the lecturer.
The program must be implemented using Python.

### Your task consists of the following components
#### Program Development
Develop a sample application that can be executed using:
Concurrent programming (e.g. threading or asynchronous execution)
Parallel programming (e.g. multiprocessing)
Your program must:
Generate or process a large volume of data/tasks
Clearly demonstrate performance differences between approaches
Example (for guidance only – DO NOT replicate):

A Sudoku generator that:
(Sudoku is exclude from Topic Choose)
Uses multiple threads/processes
Generates puzzles of varying difficulty:
100,000 very easy
10,000 easy
1,000 intermediate
100 hard
10 very hard

#### Article and User Manual on GitHub
Prepare a user manual that includes:

System requirements
Installation steps
How to run the program
Sample input/output
Screenshots (if applicable)
Source code

#### Demonstration
Video must upload to YouTube and hyperlink on GitHub (Article)
